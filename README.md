# Air Draw with Hand Tracking

This project allows you to draw in the air using your index finger.

Your finger acts as a pen and the webcam captures the drawing.

## Features

- Hand tracking using MediaPipe
- Real-time drawing with finger
- CTRL key activates drawing
- Drawing remains on screen
- Simple and lightweight project

## How It Works

1. Webcam detects your hand.
2. Index finger position is tracked.
3. When CTRL key is pressed, drawing starts.
4. When CTRL key is released, drawing stops.

Example:
Hold CTRL and draw letters in the air.
